The New folder contains a single Python script for matrix manipulation, specifically setting rows and columns to zero if any element in the matrix is zero.

File: code.py

Description: This script implements an algorithm to set entire rows and columns to zero in a matrix if any element is zero. It uses two sets to track zero-containing rows and columns, achieving O(m*n) time complexity and O(m+n) space complexity.

Dependencies: None

Code Snippet:

                
                def set_zeroes(matrix):
    rows = len(matrix)
    cols = len(matrix[0])
    zero_rows = set()
    zero_cols = set()

    for i in range(rows):
        for j in range(cols):
            if matrix[i][j]==0:
                zero_rows.add(i)
                zero_cols.add(j)

    for i in range(rows):
        for j in range(cols):
            if i in zero_rows or j in zero_cols:
                matrix[i][j]=0
    
    return matrix
