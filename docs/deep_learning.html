Overview

The deep learning folder contains eight Python scripts implementing various deep learning models using TensorFlow and Keras. The tasks range from feedforward neural networks to convolutional neural networks (CNNs) and recurrent neural networks (RNNs) for tasks like classification, digit recognition, and sentiment analysis.

File: 4th.py

Description: This script builds a CNN for digit recognition using the MNIST dataset. It preprocesses the data, constructs a model with convolutional and pooling layers, and evaluates its performance.

Dependencies: tensorflow, matplotlib

Code Snippet:

                
                import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense, Dropout
from tensorflow.keras.datasets import mnist
from tensorflow.keras.utils import to_categorical

(x_train, y_train), (x_test, y_test) = mnist.load_data()

x_train = x_train.reshape(-1, 28, 28, 1).astype('float32') / 255.0
x_test = x_test.reshape(-1, 28, 28, 1).astype('float32') / 255.0

y_train = to_categorical(y_train, num_classes=10)
y_test = to_categorical(y_test, num_classes=10)

model = Sequential([
    Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    MaxPooling2D((2, 2)),
    Conv2D(64, (3, 3), activation='relu'),
    MaxPooling2D((2, 2)),
    Flatten(),
    Dense(128, activation='relu'),
    Dropout(0.5),
    Dense(10, activation='softmax')
])

model.compile(optimizer='adam', 
              loss='categorical_crossentropy', 
              metrics=['accuracy'])
